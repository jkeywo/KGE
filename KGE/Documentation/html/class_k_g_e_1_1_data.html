Detailed Description

KGE::Data This wrapper class designed to store any type of data. To find the data types supported, please see the KGE::Data::DATA_TYPES enum values. An instance of this class can only store one type of data, thus retrieving an int as a bool is NOT supported. the getters of this class always return a pointer to the data, and if the requested type does not match the type of the stored data, NULL is returned.
